Problem 2
=========

``` r
## Set up the data fraame for problem 2
prob2_df = tibble(
  x = rnorm(1000, sd = .5),
  y = rnorm(1000, sd = .5),    
  vec_logi = x+y > 0,
  vec_num = as.numeric(vec_logi),
  vec_factor = as.factor(vec_logi)
)
```

``` r
## Scatter plot of x and y, color using logical vector
ggplot(prob2_df, aes(x = x, y = y, color = vec_logi)) + geom_point()
```
